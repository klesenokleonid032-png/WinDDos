# WinDDos

**ВНИМАНИЕ: В ИГРЕ НЕТУ АНГЛИЙСКОЙ ЛОКАЛИЗАЦИИ**

WinDDos — это проект-симулятор с набором мини-игр. Я планирую развивать игру, если будут идеи и силы. Если у вас есть предложения — пишите! (Discord - bulld0z3r_xd)

## Что есть в игре:

* **Классическое казино:** Испытай удачу! Нужно выбрать 1 число из 2. Если угадал — забираешь куш.
* **Слоты:** Классическая рулетка с тремя типами символов. Проверь свою фортуну.
* **Биржа криптовалют:** Механика рандомного курса в реальном времени. Крипта может взлететь до небес или обвалиться в ноль — успей продать вовремя!
* **Майнинг:** Самый быстрый (и немного читерский) способ фарма ДДос-коинов. 
  * **ПРЕДУПРЕЖДЕНИЕ:** Не рекомендуется людям с эпилепсией (консоль интенсивно мерцает при запуске модуля).*

## Как запустить:
1. Скачайте архив из раздела **Releases**.
2. Распакуйте папку.
3. Запустите `WinDDos.exe`.
